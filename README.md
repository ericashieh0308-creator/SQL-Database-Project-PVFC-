# SQL-Database-Project-PVFC-


## Objective
This project involves designing and implementing a relational database system for a business scenario.The goal is to demonstrate database design, data transformation, and analytical querying skills using SQL.


## Dataset
- Source: Course project dataset (CSV / JSON files)
- Includes multiple related tables such as:
  - Customers
  - Orders
  - Products
  - Sales data



## Tools
- MySQL Workbench  
- SQL (DDL & DML)  


## Methodology

### 1. Database Design
- Designed ER diagram  
- Defined relationships (1:1, 1:M, M:N)  
- Applied normalization  

### 2. Data Processing
- Imported data from CSV/JSON into staging tables  
- Transformed and inserted into business tables  

### 3. SQL Analysis
- Aggregation queries  
- JOIN operations  
- Ranking functions (RANK, DENSE_RANK)  
- Window functions (LAG)  


## Key Results
- Built a fully functional relational database with multiple linked tables  
- Successfully transformed raw data into structured business tables  
- Executed complex SQL queries including joins, aggregations, and window functions  


## Business Insights
- Identified top-performing products contributing to overall sales  
- Analyzed customer purchasing patterns to support marketing strategies  
- Demonstrated how structured data enables better business decision-making
